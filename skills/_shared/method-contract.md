# Human to Agent method contract

Methods start from a real case and direct evidence. Ask architecture-impacting questions one at a time. Record the evidence basis and cheapest probe for uncertainty; never invent a rule to make a gate pass. Deviations use plan, discovery, conservative-choice, and revisit fields. Evaluation is independent from implementation. Run `hta validate` before `hta record-change`; persistent state belongs in normative source files, not chat history.
