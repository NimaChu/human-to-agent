# Five-stage method

1. Reproduce one real task; capture input, output, chronology, judgments, manual edits, timing, usability, and initial Unknowns.
2. Stabilize a Task Contract and first Skill from evidence; rerun the original case and plan the next cases.
3. Validate the Skill on normal, boundary, and failure cases with explicit evaluators and a non-creator run.
4. Compose Goal, Skills, Context, State, Policies, Human Gates, Exceptions, local evaluators, final evaluator, stop/recovery controls, and an end-to-end trace.
5. Assess ten core and six supplemental Loop Readiness dimensions. Recommend an autonomy ceiling; the owner approves autonomy separately.

Every gate distinguishes a proven fact, a gap, and an indeterminate claim with missing evidence. New contradictions or material version changes reopen the affected assessments; stages are not calendar milestones.
